## Add repository

`helm repo add ectobit https://charts.ectobit.com`

## Install

```sh
helm install pgweb ectobit/pgweb
```

## Helmfile

```yaml
releases:
  - name: pgweb
    chart: ectobit/pgweb
    namespace: default
    values:
      - ingress:
          enabled: true
          annotations:
            kubernetes.io/ingress.class: nginx
          hosts:
            - host: pgweb.your-domain.com
              paths:
                - path: /
                  pathType: Prefix
          tls:
            - secretName: your-domain-com-tls
              hosts:
                - pgweb.your-domain.com
```
